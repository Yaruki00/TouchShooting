# TouchShooting

## 概要
画面タッチで操作を行うシューティングゲーム  
名前のまんま  
猫かわいいよ、猫

----
## ルール

![play demo](https://raw.githubusercontent.com/Yaruki00/TouchShooting/master/game.png)

画面をタップするとその方向にケーキが飛びます。  
ケーキが猫に当たると1匹5点。  
ミスショットは-1点です。  
猫は全部で20匹出現するので、最高100点です。
